# Mr. Blue Sky (2025-1)

[Acesso ao projeto no github](https://github.com/VCCorpTeam/Leitor-de-clima)

Desenvolvimento de uma ferramenta eficiente para consolidar e gerenciar dados climáticos de cidades do estado de São Paulo, permitindo a análise e geração de relatórios a partir de múltiplos arquivos CSV provenientes de diversas estações de monitoramento.

## Tecnologias

Java e JavaFX

- Linguagem e framework utilizados para o desenvolvimento da interface gráfica e lógica da aplicação.

Maven

- Ferramenta de automação de build utilizada para gerenciar dependências e facilitar a compilação do projeto.

MySQL

- Banco de dados relacional utilizado para armazenar as informações extraídas e manipuladas.

## Contribuições pessoais

### Desenvolvimento de ETL dos dados:
Desenvolvimento de funcionalidade de upload dos arquivos CSV para realizar a extração dos daodos, tranformá-los para que pudessem ser analisados e, por fim, importá-los em banco de dados.

### Desenvolvimento de interfaces da aplicação:
Estruturação e criação de interfaces de análises gráficas dos dados importados e de configurações de exbição e unidades de medida dos dados.

Aplicação possuía vistas dos dados através de filtros dinâmicos, além de possibilitar a edição de dados que apresetaram inconsistências na importação.

### Geração de relatórios:
Exportação dos dados organizados em planilhas ou em formato BoxPlot.
